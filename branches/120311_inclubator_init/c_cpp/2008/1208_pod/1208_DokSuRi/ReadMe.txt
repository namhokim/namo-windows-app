========================================================================
    WIN32 응용 프로그램 : WisdomLite 프로젝트 개요
========================================================================

응용 프로그램 마법사에서 이 WisdomLite 응용 프로그램을 만들었습니다.

이 파일에는 WisdomLite 응용 프로그램을  구성하는 각 파일에 대한
요약 설명이 포함되어 있습니다.


WisdomLite.vcproj
    응용 프로그램 마법사를 사용하여 생성한 VC++ 프로젝트의 기본 프로젝트 파일입니다.
    파일을 생성한 Visual C++ 버전에 대한 정보와
    응용 프로그램 마법사를 사용하여 선택한 플랫폼, 구성 및 프로젝트 기능에 대한
    정보가 들어 있습니다.

WisdomLite.cpp
    기본 응용 프로그램 소스 파일입니다.

/////////////////////////////////////////////////////////////////////////////
응용 프로그램 마법사에서 다음 리소스를 만들었습니다.

WisdomLite.rc
    프로그램에서 사용하는 모든 Microsoft Windows 리소스의
    목록입니다.  여기에는 RES 하위 디렉터리에 저장된 아이콘, 비트맵 및 커서가
    포함됩니다.  이 파일은 Microsoft Visual C++에서 직접 편집할 수
    있습니다.

Resource.h
    새 리소스 ID를 정의하는 표준 헤더 파일입니다.
    Microsoft Visual C++에서 이 파일을 읽고 업데이트합니다.

WisdomLite.ico
    아이콘 파일이며, 응용 프로그램의 아이콘(32x32)으로 사용됩니다.
    이 아이콘은 기본 리소스 파일인 WisdomLite.rc에 의해 포함됩니다.

small.ico
    응용 프로그램의 아이콘으로 사용되는 파일로, 16x16 크기의 작은 버전의
    아이콘이 포함되어 있습니다. 이 아이콘은
    기본 리소스 파일인 WisdomLite.rc에 의해 포함됩니다.

/////////////////////////////////////////////////////////////////////////////
기타 표준 파일:

StdAfx.h, StdAfx.cpp
    이 파일은 미리 컴파일된 헤더(PCH) 파일인 WisdomLite.pch와
    미리 컴파일된 형식(PCT) 파일인 StdAfx.obj를 빌드하는 데 사용됩니다.
