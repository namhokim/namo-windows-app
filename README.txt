[MSVC 폴더 무시할 공통 파일 설정법]
(TortoiseSVN가 필요: http://tortoisesvn.net/ )
1. Visual Studio .sln 파일이 있는 폴더를 선택.
2 오른쪽 마우스를 눌러 [TortoiseSVN] - [Properties] 선택
3. Import... 버튼을 누르고, template/msvc-svn_ignore.svnprops 를 선택
 (파일: *.ncb *.suo *.user  / 폴더: Debug Release)
