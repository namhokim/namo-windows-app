1. 프로그램의 시작과 종료
 - 자동실행 : 프로그램은 [시작]메뉴에 '시작프로그램' 바로가기를 등록
 - 수동실행 : 사용자에 의해 프로그램의 실행(.exe 혹은 바로가기)
 - 수동종료 : [파일]-[종료] 메뉴 선택시
 - 자동종료 : 시스템 종료시(통지 받을 경우) 상태 저장후 정리/종료

2. 프로그램 중복 제한
 - 프로그램은 한번에 하나의 인스턴스(프로세스)가 생성되어야 한다.
 - 기존에 프로그램이 실행이 되어 있을 경우는 트레이/최소화된 창이 활성화되어
   화면에 표시되어야 한다.

3. 파일시스템
 - 외부 파일을 이용해서 작업 내역을 기록한다.
 - 파일의 형태는 일반 텍스트 파일이다.
 - 지원 파일 인코딩은 ANSI와 UTF-16LE이 있다.
   지원 형태는 파일 처음 2바이트의 BOM의 유무로 한다. (FF FE)

 - 외부 파일은 실행파일이 있는 곳에 생성되며,
   파일명은 YYYYMMDD.txt로 하고 각 의미는 연(4자리), 월(2자리), 일(2자리)이다.
 - 만약 월과 일이 1자리일 경우 앞에 0을 패딩하여 고정된 길이로 사용한다.

 - 파일포맷
   6개의 필드가 있고 각각은 아래와 같다.
   항목명        | 타입      | 형식                | 기타
   ---------------------------------------------------------------------
   시퀀스(seq #) | 숫자      | N                   | 1부터 시작하며 부호 없는 정수형
   작업          | 텍스트    |                     | 유니코드 문자, 길이 제한은 없음
   그룹          | 텍스트    |                     | 유니코드 문자, 길이 제한은 없음
   시작          | 날짜/시간 | YYYY/MM/DD HH:mm:SS | 지역시간을 사용(local time)
   끝            | 날짜/시간 | YYYY/MM/DD HH:mm:SS | 지역시간을 사용(local time)
   수행시간      | 시간      | [H]HH:mm:SS         | 시작>끝 시간인 경우는 마이너스 값이 될 수 있음

   각 필드의 구분은 수평탭(Vertical Tab, '\t')으로 한다.

 - 기록시기(Write)
   작업이 하나가 수행되면 파일 덮어쓰기를 수행한다.
   파일이 없을 경우는 생성
   파일이 있을 경우는 덮어씀
     (쓰기 불가일 경우는 메시지 창을 띄움, 에러내용)
     (기록 못한 내용은 다음 번에 기록을 하게끔 함)
   
 - 로딩시기(Read)
   프로그램이 시작시 파일의 존재를 체크하여, 있을 경우 프로그램 메모리에 불러들임

4. 작업기록
 - 작업 기록은 '작업 이름'과 '그룹'을 입력하는 창에서 수행
 - 기록 명령은 메뉴 혹은 단축키(액셀러레이터)에서 가능하다.
   메뉴 : [작업]-[추가]
   단축키 : Insert / Alt-S,A / +
 - 자주 사용했던 (작업,그룹)은 MRU(Most Recently Used)로 최대 10개까지 관리한다.
   이후 메뉴 하위에 추가됨(-로 구분하여)
   관리 정보는 레지스트리에 관리됨
 - 작업이 추가가 되면 '외부 파일'에 기록을 시도함
   (시도라고 표현한 것은 디스크 액세스 관련해서 실패 가능성도 있으므로)

5. 레지스트리
 - 프로그램의 정보를 시스템에 저장을 함
 - 저장 정보는 아래와 같은
   작업 MRU 목록
	HKEY_CURRENT_USER\Software\SRT\Jobs\jobCnt
	HKEY_CURRENT_USER\Software\SRT\Jobs\job1
	 ~
	HKEY_CURRENT_USER\Software\SRT\Jobs\job#
   UI관련(윈도우 크기 및 위치)
	HKEY_CURRENT_USER\Software\SRT\Settings\Position\Left
	HKEY_CURRENT_USER\Software\SRT\Settings\Position\Top
	HKEY_CURRENT_USER\Software\SRT\Settings\Position\Right
	HKEY_CURRENT_USER\Software\SRT\Settings\Position\Bottom
 - 자동시작 관련(항목 8참조)

6. UI
 - 아이콘은 바뀐 회사 로고를 사용
 - 기본적으로 Listview가 뷰인 Windows Applicaton으로 작성
 
7. 합산기능(2.1.0.2)
 - 복수의 리스트 선택시 하단의 상태바에 아래의 내용이 표시된다.
 - 선택한 항목수, 소요시간의 합계
  예) n 개체가 선택됨 | 0:00:00

8. 자동실행기능
 - 메뉴에서 체크를 하면 자동시작 목록에 등록시킨다. (항목5참고)
 - 토글 '자동실행/미실행'이 가능해야 한다.
